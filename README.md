# shop-example

쇼핑몰 Side Project
====================
사용 기술 및 프레임 워크
---------
Spring Boot, Spring Security, JWT, JPA (Hibernate), React.js, Mysql

기능 목록
---------
* 로그인
>       회원가입
>       로그인
>       로그아웃

* 회원정보
>       회원정보 조회/수정
 
* 제품정보
>       제품 list
>       제품 상세
>       댓글
>       주문/결제

* ADMIN
>       회원 List
>       제품 등록
>       제품 삭제

* 마이페이지	
>       주문목록
>       주문취소
>       취소목록
>       주문상세보기
>       장바구니
